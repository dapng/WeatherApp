Погодное приложение

Создано с использованием языка программирования Python.
Для построения приложения использовалась база данных SQL.
Для определения погоды используется API (программный интерфейс приложения) от openweathermap.
Приложение запоминает введенный город и отображает актуальные погодные данные.
Управление списком городов осуществляется по ссылке /admin, данные для входа admin, можно изменять и удалять и добавлять города из СУБД.
